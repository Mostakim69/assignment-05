Question-2: What is the difference between innerHTML, innerText, and textContent ?

১ ... innerHTML,,
এটি HTML ট্যাগসহ সম্পূর্ণ কনটেন্ট সেট করে।
যদি এলিমেন্টের মধ্যে HTML থাকে, তাহলে এটি HTML প্রসেস করবে।
২... innerText,,
এটি শুধুমাত্র প্রদর্শিত (visible) টেক্সট সেট করে।
Hidden (display: none) এলিমেন্টের টেক্সট নেবে না।
৩... textContent,,
এটি সম্পূর্ণ টেক্সট কনটেন্ট সেট করে, HTML প্রসেস ছাড়াই।
Hidden এলিমেন্টের টেক্সটও ধরে।

Question-3: What is event delegation in the DOM?

Event Delegation হল JavaScript-এর একটি স্কিল ইভেন্ট হ্যান্ডলিং টেকনিক, , যেখানে একটি Parent এলিমেন্টের ওপর ইভেন্ট Listener সেট করা হয় এবং সেই Parent এলিমেন্টের মাধ্যমে তার Child এলিমেন্টগুলোর ইভেন্ট পরিচালনা করা হয়।

Question-4: What is event bubbling in the DOM?

যদি একটি Child এলিমেন্টে কোনো ইভেন্ট ঘটে, তাহলে সেই ইভেন্ট Parent, Grandparent পর্যন্ত পৌঁছে যায় যদি সেখানে একই ইভেন্টের লিসেনার থাকে।
